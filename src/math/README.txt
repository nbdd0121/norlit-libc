The directories are categorized according to C11 categories
trig : Trigonometric functions
hyp  : Hyperbolic functions
exp  : Exponential and logarithmic functions
pow  : Power and absolute-value functions
err  : Error and gamma functions
rint : Nearest integer functions
rem  : Remainder functions
manip: Manipulation functions
max  : Maximum, minimum, and positive difference functions
fma  : Floating multiply-add
